# 微信小程序学习日记（1）
## 微信小程序 --- 事件绑定


#### 事件类别：

* tap：点击事件；

* longtap：长按事件；

* touchstart：触摸开始；

* touchend：触摸结束；

* touchcansce：取消触摸；

#### 事件绑定：

* bind绑定；

* catch绑定；（能阻止事件冒泡）


>例如：绑定点击事件 bindtap

page.wxml 文件
``` html
<button bindtap="btnclick">click</button>
```
page.js 文件
``` javascript
//获取应用实例
const app = getApp()
Page({
  data: {
    text: "你好",
    onOff: false
  },
  btnclick:function(){
      console.log('123');
      var onOff = this.data.onOff;
      this.setData({text:"hello",onOff:!onOff});
  }
})
```
